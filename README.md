# Jogo-da-Velha

Projeto Jogo da velha em c++. Esse projeto utiliza a função stoi() do c++ 11, logo, será necessário habilitá-lo em seu compilador, ou execute o arquivo exe que o acompanha.

Tutorial para habilitar:
